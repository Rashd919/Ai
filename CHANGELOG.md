## نوتات الإصدار

### كيفية الترقية من 1.0.0 إلى 2.0.0

1. **استنساخ الإصدار الجديد:**
```bash
git pull origin main
```

2. **تثبيت المكتبات الجديدة:**
```bash
pip install -r requirements.txt
```

3. **تحديث المفاتيح السرية:**
تأكد من إضافة جميع المفاتيح المطلوبة في `.streamlit/secrets.toml`

4. **إعادة تشغيل التطبيق:**
```bash
streamlit run app.py
```
