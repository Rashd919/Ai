## 🚀 البدء السريع

### المتطلبات

- Python 3.8 أو أحدث
- pip أو pip3
- توكن Groq API
- توكن Telegram Bot (اختياري)

### التثبيت

1. **استنساخ المستودع:**
```bash
git clone https://github.com/Rashd919/Ai.git
cd Ai
```

2. **تثبيت المكتبات:**
```bash
pip install -r requirements.txt
```

3. **إعداد المفاتيح السرية:**
قم بتحرير ملف `.streamlit/secrets.toml` وأضف مفاتيحك:
```toml
GROQ_API_KEY = "your_groq_api_key"
TELEGRAM_BOT_TOKEN = "your_telegram_token"
TELEGRAM_CHAT_ID = "your_chat_id"
```

4. **تشغيل التطبيق:**
```bash
streamlit run app.py
```

سيفتح التطبيق على `http://localhost:8501`
